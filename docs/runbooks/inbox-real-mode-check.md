# 口コミ・コメント受信箱（実機確認）手順

値は一切記載せず、クリック/入力単位のチェックリストだけを記載します。

## 目的
- 口コミ・コメント受信箱で「取得」「返信」が実機で動くことを確認する

## 事前準備
- [ ] `/admin/diagnostics` で必須の環境変数が「設定済み」になっている
- [ ] `/admin/provider-health` で接続状態が確認できる
- [ ] 対象の店舗が作成済み
- [ ] Google/Meta の連携サービスが接続済み

## 1. Google口コミ（取得/返信）
- [ ] `/app/reviews` を開く
- [ ] フィルタで「Google口コミ」を選ぶ
- [ ] 一覧に口コミが表示されることを確認する
- [ ] 任意の口コミを開き、返信欄に本文を入力する
- [ ] **返信を送信** を押して成功メッセージが出ることを確認する

## 2. SNSコメント（Facebook/Instagram）
- [ ] `/app/reviews` を開く
- [ ] フィルタで「SNSコメント」を選ぶ
- [ ] 一覧にコメントが表示されることを確認する
- [ ] コメントを開き、返信欄に本文を入力する
- [ ] **返信を送信** を押して成功メッセージが出ることを確認する

## 3. 失敗時の分岐（最短）
- **連携サービスが未接続**
  - `/app/setup` から連携サービスをつなぐ
- **権限が不足している**
  - `/admin/diagnostics` で権限の不足項目を確認
  - 必要な権限を追加して再接続
- **店舗の紐付けが未完了**
  - `/app/locations/[id]` で紐付けを完了する
- **一時的な制限**
  - 時間をおいて再確認する

## 注意
- トークン/シークレット/署名URLの完全値は共有・記録しない
