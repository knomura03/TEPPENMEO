# 投稿テンプレの準備手順

## 目的
- 投稿テンプレを用意して、日々の投稿を短時間で作成できるようにする

## 前提
- owner / admin 権限でログイン
- Supabaseが設定済み
- `0009_post_templates.sql` が適用済み

## クリック/入力手順（最短）
### 1) マイグレーションを適用する（CLI推奨）
1) ターミナルで `supabase db push` を実行する
2) `/admin/diagnostics` で **post_templates** が「適用済み」になっていることを確認する

### 2) テンプレを作成する
1) `/app/post-templates` を開く
2) **テンプレを作成** を押す
3) テンプレ名と本文を入力する
4) 投稿先（Google / Facebook / Instagram）を選ぶ
5) **保存** を押す

### 3) 店舗詳細でテンプレを使う
1) `/app/locations/<id>` を開く
2) 投稿セクションで **テンプレを選ぶ** を開く
3) 作成したテンプレを選択する
4) 本文が反映されたことを確認して投稿する

## よくある詰まりと対処
- **テンプレ一覧が表示されない**
  - `0009_post_templates.sql` が未適用の可能性があります。`/admin/diagnostics` を確認してください。
- **保存できない**
  - `SUPABASE_SERVICE_ROLE_KEY` 未設定の可能性があります。
- **テンプレが反映されない**
  - テンプレの本文が空になっていないか確認してください。

## 注意
- トークン/シークレット/署名URLの完全値は共有・記録しない
