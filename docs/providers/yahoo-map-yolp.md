# Yahoo! YOLP

## 前提
- [ ] Yahoo!デベロッパー登録
- [ ] アプリケーション作成権限

## 画面操作の手順（クリック単位）
- [ ] Yahoo!デベロッパーネットワークにログイン
- [ ] **アプリケーション管理** → **新規作成**
- [ ] アプリ名/利用API（ローカルサーチ）を設定
- [ ] **アプリケーションID（App ID）** を取得

## 設定値
- TEPPEN MEOはローカルサーチのみ使用

## 必要な環境変数
- `YAHOO_YOLP_APP_ID` : YOLP App ID

## 動作確認手順
- [ ] `/app/locations` の検索フォームで **Yahoo! YOLP** を選択
- [ ] キーワード検索で候補が表示されれば成功

## よくある詰まり
- App ID未設定: `YAHOO_YOLP_APP_ID` を確認
- 文字コード/特殊文字で失敗: 短い日本語キーワードで再試行
